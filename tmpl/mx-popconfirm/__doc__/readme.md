### API

#### options
| 参数 | 说明 | 类型 | 默认值 |
| -------- | -------- | -------- | -------- |
| content    | 提示内容 | string | 确认执行该操作？ |
| enterText     | 确定按钮内容 | string | 确定  |
| cancelText     | 取消按钮内容 | string | 取消  |
| placement | 提示框在目标的方位，top,bottom,left,right | string | left |
| align | 提示框与目标的对齐方式，top,bottom,left,right | string | &nbsp; |
