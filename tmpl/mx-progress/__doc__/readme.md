## 进度条组件

#### 使用

<a href="https://magix-components.github.io/magix-components/#!/mx-progress/index" target="_blank">在线示例</a>

```html
<mx-progress
    value="0.36"
    class="fl"/>
```


### API

#### options
| 参数 | 说明 | 类型 | 默认值 |
| -------- | -------- | -------- | -------- |
| value    | 设置百分比值，0-1之间的小数 | number | 0 |
| fixed    | 小数位 | number | 0 |


#### method

| 方法名 | 说明 | 入参 | 返回值 | 说明 |
| -------- | -------- | -------- | -------- | -------- |
| val | 设置或获取值 | number | number | &nbsp; |

> method调用方法： $('#id').invokeView('val',[20]);
