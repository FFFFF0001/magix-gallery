### API

#### options
| 参数 | 说明 | 类型 | 默认值 |
| -------- | -------- | -------- | -------- |
| list    | 列表数组，可为简单数组或[1,2,3]或对象数组，如[{id:1,text:'a'}] | array |  |
| textKey     | 当列表中的项为对象时，渲染text时读取的key | string |  |
| valueKey     | 当列表中的项为对象时，渲染value时读取的key | string |  |
